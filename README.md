# Banking Management System SQL Project


This project is a **SQL-based Banking Management System**. It simulates a bank's database with accounts, transactions, and reports.


## Features
- Account creation and management
- Transaction tracking (Deposit, Withdrawal, Transfer)
- Balance calculation per account
- Fraud detection queries
- Monthly statements generation


## How to Use
1. Import `schema.sql` to create the database and tables.
2. Import `insert_data.sql` to populate sample accounts and transactions.
3. Run `queries.sql` to see balance calculations, detect fraud, and generate monthly statements.


## Folder Structure
- `schema.sql` : Database & table creation
- `insert_data.sql` : Sample data
- `queries.sql` : Advanced queries
- `docs/info.yaml` : Project metadata
